# Ai-Link_A.O.Smith

此集成允许您在 Home Assistant 中控制 Ai家智控（A.O.Smith） 热水器。

## 功能

- 读取热水器状态
- 显示温度、水流量等传感器数据
- 自动发现设备

## 安装

1. 通过 HACS 添加此自定义仓库
2. 重启 Home Assistant
3. 通过 UI 配置集成

## 配置

您需要通过抓包获取以下参数：
- access_token
- user_id  
- family_id
- device_id (可选)
- mobile (可选)
